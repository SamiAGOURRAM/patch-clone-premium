## üìù Step-by-Step: Customize a Section

### Example: Change Stats Section Background

1. **Go to Sanity Studio**: https://auroraa.sanity.studio
2. **Click "Couleurs par Section"**
3. **Click "+" to create new** (or edit existing)
4. **Select "Stats Section"** from dropdown
5. **Set "Couleur de Fond"**: `220 45% 18%` (dark blue)
6. **Set "Couleur du Texte Principal"**: `0 0% 100%` (white)
7. **Click "Publish"**
8. **Refresh your website** - Stats section now has custom colors!

### Example: Customize Hero Section Buttons

1. **Create/edit "Hero Section" colors**
2. **Set "Couleur des Boutons - Fond"**: `220 70% 50%` (blue)
3. **Set "Couleur des Boutons - Texte"**: `0 0% 100%` (white)
4. **Publish** - Hero section buttons are now blue!

## üí° Tips

1. **Create one color setting per section** - Each section can have its own colors
2. **Leave fields empty** - Uses default colors if not specified
3. **Mix and match** - Some sections can use defaults, others custom colors
4. **HSL format** - Always use: `Hue Saturation% Lightness%` (no hsl() wrapper)
