## üñºÔ∏è Working with Images

### Uploading Images

1. Click on any image field
2. Click **"Upload"** or drag and drop an image
3. Recommended formats: JPG, PNG, WebP
4. Recommended size: 
   - Hero images: 1920x1080px or larger
   - Profile photos: 400x500px
   - Logos: 200x80px
5. After upload, you can:
   - Crop the image (click and drag)
   - Add alt text for accessibility
   - Replace the image

### Image Best Practices

- Use high-quality images
- Optimize file sizes (Sanity does this automatically)
- Use descriptive filenames before uploading
- For logos, use transparent backgrounds (PNG)

## ‚úÖ Publishing Changes

**Important:** After making any changes, you must click **"Publish"** for changes to appear on your website.

1. Make your edits
2. Review the changes in the preview (if available)
3. Click the **"Publish"** button in the top right
4. Changes will appear on your website within a few seconds to a few minutes

## üîÑ Draft vs Published

- **Draft**: Changes saved but not visible on the website
- **Published**: Changes are live on the website

You can save drafts and publish later, or publish immediately.

## üóëÔ∏è Deleting Content

1. Open the content item you want to delete
2. Click the **"Delete"** button (usually in the top right or bottom)
3. Confirm the deletion
4. **Note:** Deleted content cannot be easily recovered, so be careful!

## üîç Tips & Best Practices

1. **Save Often**: Use "Publish" to save your work
2. **Preview First**: Review changes before publishing if possible
3. **Order Matters**: Use the "Ordre d'affichage" field to control display order
4. **Test Changes**: After publishing, check your website to see the changes
5. **Keep Backups**: Important content should be backed up (your developer can help)

## üÜò Troubleshooting

### Changes Not Appearing

- Make sure you clicked **"Publish"** (not just saved)
- Wait a few minutes for changes to propagate
- Clear your browser cache (Ctrl+F5 or Cmd+Shift+R)
- Check if you're looking at the correct page

### Can't Upload Images

- Check file size (should be under 10MB)
- Try a different image format (JPG or PNG)
- Check your internet connection

### Lost Changes

- Check if you saved as draft (look for "Unpublished changes")
- Check the revision history (if available)
